# Apache Hudi (apache-hudi)
Apache Hudi is a data lake platform that provides incremental data processing primitives including upserts and incremental queries. It manages storage of large analytical datasets on distributed file systems with ACID transactions.

## APIs

### Apache Hudi
Hudi provides Java APIs for write operations (upserts, inserts, deletes) with timeline and compaction APIs, plus integration APIs for Spark, Flink, and Hive for reading and writing Hudi tables.
